This is an image loading screen.
To use this, create an element with class content-container. that acts as the image container
Let say we use a <div> element. <div class="content-container">

</div> Then, wrap image inside an element with class content-container.
Put this code below inside content-container element. <div class="img-loader-wrapper">
<span class="img-loader"><span class="img-loader-inner"></span></span>
</div> After that, you can put the image element inside content-container element.
Don't forget to make the content-container element's width be the same as the image element's width.
